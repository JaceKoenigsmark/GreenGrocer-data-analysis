# GreenGrocer Business Analytics Project

## Overview
This project is a complete **Business Analytics simulation** for a fictional grocery chain called **GreenGrocer**. It uses Python to generate realistic store, customer, sales, and operational datasets, then performs descriptive, diagnostic, and predictive analytics.

The goal of the project is to help management understand business performance, identify opportunities, forecast growth, and make smarter decisions. :contentReference[oaicite:0]{index=0}

### Descriptive Analytics

- Data visualizations

## Dataset Simulation

The project creates realistic synthetic data for:

### Stores
Five Florida locations:
- Tampa
- Orlando
- Miami
- Jacksonville
- Gainesville

### Departments
- Produce
- Dairy
- Bakery
- Grocery
- Prepared Foods

### Customers
5,000 simulated customers with:
- Age
- Gender
- Income
- Spending behavior
- Loyalty tiers
- Preferred store

### Sales Data
Daily transactions across all stores and departments for one full year.

## Key Analyses Included

### 1. Sales Performance
Calculates:
- Total sales
- Total profit
- Mean sales
- Median sales
- Standard deviation
- Profit margin

### 2. Visual Reports
Creates charts for:
- Sales by store
- Sales by department
- Monthly sales trends
- Customer segment distribution
- Seasonal demand patterns

### 3. Correlation Analysis
Measures how factors like:
- Square footage
- Staff count
- Marketing spend
- Customer satisfaction

influence annual sales.

### 4. Forecasting
Uses regression and moving averages to predict:
- Future department sales
- Store performance

### 5. Recommendations
Generates actionable strategies such as:
- Improving weak departments
- Expanding strong store practices
- Better staffing during peak months
- Targeting high-value customers
