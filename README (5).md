# GreenGrocer Business Analytics Project

## Overview
This project is a complete **Business Analytics simulation** for a fictional grocery chain called **GreenGrocer**. It uses Python to generate realistic store, customer, sales, and operational datasets, then performs descriptive, diagnostic, and predictive analytics.

The goal of the project is to help management understand business performance, identify opportunities, forecast growth, and make smarter decisions.

## Features

### Descriptive Analytics
- Sales performance statistics
- Profit analysis
- Store comparisons
- Department comparisons
- Customer segment analysis
- Data visualizations

### Diagnostic Analytics
- Correlation analysis
- Operational factor impact on sales
- Store efficiency comparison
- Seasonal trend analysis

### Predictive Analytics
- Store sales prediction model
- Department sales forecasting
- Growth trend estimation

### Business Intelligence
- Profit opportunity identification
- Strategic recommendations
- Executive summary generation

## Technologies Used

- Python 3
- Pandas
- NumPy
- Matplotlib
- SciPy
